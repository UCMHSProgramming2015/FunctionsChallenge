# Name: Quadratic Formula

## Examples:
QFormula(2,14,3)
QFormula(3,23,13)

## Description:
Finds a zero of the equation. This function only calculates the quadratic formula with addition (-b + sqrt... not -b - sqrt...). The paramteres are the constant values of the equation.

## Syntax:
QFormula(a,b,c)

##Parameters: 
a       float:value a
b       float:value b
c       float:value c

##Returns:
float
