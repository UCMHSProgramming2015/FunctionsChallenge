# Name: mathOperations()

## Examples:
void draw() {
  println(mathOperations(5));
}

float mathOperations(float number) {
  return number/2*7 +5;
}

## Description:
Divides a any number by two, multiplies it by seven, and adds five to the product.

## Syntax:
mathOperation(n)

##Parameters: 
n           float: value to be evaluated

##Returns:
float
