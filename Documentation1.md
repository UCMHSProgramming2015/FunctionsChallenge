## Name: 
areaOfCircle()

## Examples:
  float areaOfCircle(){
    float r= 6;
    float A= PI*sq(r);
    return A;
  }

## Description: 
Enter radius, and the area of the circle is calculated.

## Syntax:
  float areaOfCircle(){
    float r= ?;
    float A= PI*sq(r);
    return A;
  }

##Parameters: 
? is radius ;
A is calculated area

##Returns:
Area displayed in text
