## Description:
With this function you can enter any farenheit value and it will print under the command line the same value, but in kelvin

## Syntax:
 ConvertFarenheitToKelvin (Fdeg); 

##Parameters: 
Fdeg: The farenheit value you want to convert

##Returns:
int
