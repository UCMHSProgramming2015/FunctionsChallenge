# Name: Area Function

## Examples:
float b= 100;
float h= 50;

## Description:
With a given base and height, the function Area provides just that, the area. 

## Syntax:

float b = 100;
float h =50;

void setup() {
  size(800, 600);
}

void draw() {

  println(Area(b, h));
}

float Area(float b, float h) {
  float Area;
  Area= b* h;
  return(Area);
}
##Parameters: 
println(Area(b, h)); -provides the area of  b and h

  return(Area); - will give the value of area
