# Name: 
float volumeOfSphere()

## Examples:
 float volumeOfSphere(){
    float r= 6;
    float V= 4/3*PI*r*r*r;
    return V;
  }

## Description:
Enter radius, and calculated volume of the sphere will be displayed

## Syntax:
  float volumeOfSphere(){
    float r= ?;
    float V= 4/3*PI*r*r*r;
    return V;
  }

##Parameters: 
? = radius to enter ; V is calculated volume

##Returns:
Volume (V) is displayed
