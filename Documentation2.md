# Name: function  

## Examples:
function (2, 15, 20, x)

## Description:
Returns the value of a trinomial(y) when a, b, c, and x are input into the function.  

## Syntax:
float function (float a, float b, float c, float x) {  
   float y = sq(a*x)+(b*x)+c;  
   return y;  
}

##Parameters: 
Float a
Float b
Float c
Float x

Function is a float

##Returns:
Returns numeric data
