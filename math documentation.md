Name: 
findKelvin (from Celsius)

Examples:

float findKelvin (float celsius){
  float kelvin= celsius+273;
  return kelvin;
}

Description:
finds the Kelvin (adds 273) for the value of Celsius inputed. 

Syntax:
kelvin(celsius);

Parameters: 
celcius	int or float, the input value of celsius

Returns:
kelvin
